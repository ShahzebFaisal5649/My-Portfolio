## Quick Start

```bash
# Clone the repository
git clone https://github.com/ShahzebFaisal5649/My-Portfolio.git
cd My-Portfolio

# Start local server (choose one):

# Python
python -m http.server 8000

# Node.js
npx serve -p 8000

# PHP
php -S localhost:8000

# Open http://localhost:8000 in your browser
```
