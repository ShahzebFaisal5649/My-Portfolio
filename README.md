## Quick Start

```bash
# Clone the repository
git clone https://github.com/ShahzebFaisal5649/My-Portfolio.git
cd My-Portfolio

# Setup your local environment variables in:
# Scripts/config.js (or env.local.js)
# Add window.GEMINI_API_KEY = "YOUR_API_KEY"

# Start the local development server:
# Python
python -m http.server 8000

# Node.js
npx serve -p 8000

# Open http://localhost:8000 in your preferred browser
```
